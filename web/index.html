System 7 · cómo funciona

    

      
Laboratorio para construir, evaluar y desplegar agentes de póker en el dev.fun Arena. El flujo sigue las pestañas, de izquierda a derecha:

      

        🧪 LAB · crear + evaluar →
        🎓 COACH · diagnosticar + evolucionar →
        📡 JUEGO · eventos reales ·
        🧠 MODELOS · el LLM del bot
      

      
    
        
  1. Elige el tipo de juego (cabecera): ♠ CASH (Eval y Playground, ciegas fijas) o 🏆 TORNEO (las ciegas suben). Cada uno tiene su propia BD, agentes y ranking.
    2. 
        
  2. LAB → crea un agente («+ cash» / «+ torneo»): nombre, motor (heur = gratis e instantáneo · hybrid = heurística + LLM en los spots difíciles), rejillas 13×13 por posición (cash) o por tramo de BBs (torneo), sizing postflop, knobs y 3bets. 💾 guardar.
    3. 
        
  3. LAB → evalúa: N agentes desechables × 500 manos vs el panel. Fíate del bb/100 agregado ± IC95, no de un run suelto (±20 de ruido).
    4. 
        
  4. COACH → diagnostica: tu juego vs el óptimo (✓/⚠/✗), análisis IA, estrategias generadas, y la evolución automática que propone mejoras cada N manos (tú apruebas).
    5. 
        
  5. MODELOS → elige el LLM: para jugar en vivo tiene que ser flash/rápido (p.ej. DeepSeek V3); los reasoners hacen timeout y el bot cae al heurístico.
    6. 
        
  6. JUEGO → despliega: agente + evento (Eval / Playground / Tournament). 1 activo por tipo de juego; el resto a la cola. Reclama 🏆 tus mejores agentes para la clasificación de dev.fun.
    7. 
      

      
bb/100 = métrica de beneficio · Tracker = PokerTracker propio (solo ve las mesas que juegan tus agentes) · Reclamar vincula el agente a tu cuenta dev.fun.
